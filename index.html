Mini Mario Prototype

A LÖVE 2D Platformer Adventure

Jump, run, and explore in this custom-built platformer engine. Featuring classic mechanics, power-ups, and a boss battle!

Download Game

Requires LÖVE framework to run.

Features

  • Classic Platforming: Precise physics with variable jump height and momentum.
  • Power-ups: Collect Red Mushrooms for speed and Green Mushrooms to grow in size.
  • Enemies: Stomp on Goombas, avoid Piranha Plants, and defeat the Boss.
  • Dynamic Levels: Explore overworld platforms and delve into underground pipe levels with lava.
  • Combat: Find the hidden gun block to take down enemies from a distance.

Controls

← / → or A / D
Move Left / Right
Z or Space
Jump
X or Shift
Run / Sprint
R
Restart Level

How to Run

  1. Install LÖVE (love2d.org) for your operating system.
  2. Download the source code or the .love file.
  3. Drag the folder onto the LÖVE application, or run love . in your terminal.
